# Label

**Import:** `import { Label } from 'vlite3'`

### Props

| Prop    | Type     | Default | Description                   |
| :------ | :------- | :------ | :---------------------------- |
| `for`   | `string` | —       | ID of the target form element |
| `class` | `string` | —       | Custom CSS class              |

### Usage

```vue
<Label for="email">Email Address</Label>
<Input id="email" />
```
